# 🍎 Blox Fruit Auto Fruit Sniper

A Roblox Lua script focused on automatically finding, grabbing, and storing spawned fruits in **Blox Fruits**.

## ✨ Features

- Continuous fruit scanning
- Auto-fly (tween) to nearest fruit
- Auto-pickup + auto-store with retries
- Auto server hop after each cycle
- In-game live status GUI + action log
- Fruit ESP overlay
- Anti-AFK support
- No-clip movement while navigating

## ⚙️ Configuration

Main settings are at the top of:

- `BloxFruit_AutoFruit.lua`

Available toggles include:

- `AutoFruitSniper`
- `FruitESP`
- `TweenSpeed`
- `StoreRetries`
- `HopDelay`
- `ScanInterval`
- `AntiAFK`

## 🚀 Usage

1. Open **Blox Fruits**.
2. Execute the script from `BloxFruit_AutoFruit.lua` using your preferred Roblox Lua executor.
3. Watch the on-screen GUI for status updates and logs.
